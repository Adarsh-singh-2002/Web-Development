priority of css implementation
!important > InlineCSS > InternalCSS > ExternalCSS
!important takes up the highest precendence. If we have some color assigned in externalCSS and want it to override the inlineCSS we could add !important at the end of the characteristics.
Here, we need to know one thing that in the internal and external css the one which is loaded at last would have the higher precedence.
Also if multiple style files are included than also the one loaded last would have the higher precedence.
